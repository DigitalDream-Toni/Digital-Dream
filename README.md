# Digital Dream Portfolio

Advanced, fully responsive portfolio for a web designer, developer, and graphic designer.

## Features
- Dark gradient theme with modern typography
- Animated preloader and welcome side notification
- Subtle 3D tilt effects and scroll reveal
- Responsive layout for desktop and mobile
- WhatsApp and Email quick contact buttons

## Tech Stack
- HTML
- CSS
- JavaScript

## How To Run
1. Open `index.html` in your browser.
2. Edit text and sections directly in `index.html`.
3. Update styles in `styles.css` and behaviors in `script.js`.

## Customize
- Replace text, stats, and services in `index.html`.
- Adjust colors and gradients in `styles.css`.
- Update motion effects in `script.js`.
